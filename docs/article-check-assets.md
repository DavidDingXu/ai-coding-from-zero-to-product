# Article And Asset Check

Use this checklist when a lesson references code, commands, screenshots, or diagrams from this repository.

## Check Items

- Code module path exists relative to the repository root.
- Commands in the lesson are present in the module README or `package.json`.
- Screenshots or diagrams are available to the publishing workflow.
- Verification command was rerun and recorded in `process/07-verification.md`.
- `process/` notes match the behavior described in the lesson.
- Real screenshots do not expose private data; illustrative diagrams are not described as live product screenshots.

## Boundary

This repository should not depend on private article draft folders. Keep reusable verification evidence inside the module's `process/` files and keep image filenames descriptive when they are referenced by a lesson.
